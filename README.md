# BlockGen

Утилита для преобразования таблиц `.xls` или `.xlsx` в набор HTML-блоков. Первая строка таблицы интерпретируется как заголовки столбцов, а каждая последующая строка превращается в отдельный блок с перечислением полей.

## Установка

```bash
npm install
```

## Использование

```bash
node src/index.js путь/к/таблице.xls [путь/к/результату.html]
```

- Если путь к выходному файлу не указан, HTML выводится в стандартный поток вывода.
- Если в таблице нет заголовков, они будут автоматически созданы (`Столбец 1`, `Столбец 2` и т. д.).

## Пример

```bash
node src/index.js ./data/example.xls ./output.html
```

Результат — страница с блоками, где каждый блок соответствует строке таблицы и содержит пары «название столбца — значение». Стили встроены в документ, поэтому итоговый файл можно открыть в браузере без дополнительных ресурсов.
